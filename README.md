# TokenTires
Project 3 for Rice University FinTech Bootcamp

Goal: Program a smart contract by using Solidity and put it on the Ethereum network. The contract will be used by Tire manufacturer's to pay out distributors and suppliers. 

Level one: is a TieredpaymentSplitter contract. This will distribute different percentages of Ether to the Tire company's suppliers and distributors at different tiers/levels.
